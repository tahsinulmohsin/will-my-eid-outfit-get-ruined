# ☪️ Will My Eid Outfit Get Ruined?

> _The only weather app that truly understands your Eid ul-Fitr priorities._

You've spent weeks picking the perfect outfit. The kurta is pressed, the shoes are spotless, and the colour coordination is *chef's kiss*. But there's one enemy you can't fight with fashion: **the weather**.

This app checks your local weather in real-time and gives you a brutally honest, screen-filling **YES** or **NO** — along with hilariously sarcastic outfit advice you didn't ask for (but definitely needed).

---

## ✨ Features

- **📍 Auto Location Detection** — Uses the browser Geolocation API. No manual input needed.
- **⛅ Real-Time Weather** — Fetches live conditions (temp, rain, humidity, wind) from the Open-Meteo API.
- **🔤 Giant YES / NO Verdict** — A massive, impossible-to-miss answer fills your screen.
- **😂 Sarcastic Outfit Advice** — Weather-specific roasts and encouragements for your Eid drip.
- **🎨 Dynamic Backgrounds** — Vibrant greens & golds for clear skies; moody grays & blues when it's gloomy.
- **✨ Smooth Transitions & Particles** — Animated backgrounds, floating particles, and pop-in effects.
- **📱 Fully Responsive** — Looks stunning on phones, tablets, and desktops.

## 🚀 How to Run Locally

No build tools, no dependencies, no drama.

1. **Clone** (or download) this repository:
   ```bash
   git clone https://github.com/your-username/will-my-eid-outfit-get-ruined.git
   ```
2. **Open** `index.html` in your browser:
   ```bash
   # macOS
   open index.html

   # Windows
   start index.html

   # Linux
   xdg-open index.html
   ```
3. **Allow location access** when prompted — that's it!

> **Note:** A live internet connection is required for Tailwind CDN, Google Fonts, and the weather API.

## 🌐 Deployment

This project is a **single static HTML file**, making it perfect for instant deployment.

### Vercel (Recommended)

1. Push this repo to GitHub.
2. Import the repo on [vercel.com/new](https://vercel.com/new).
3. Vercel auto-detects static files — no configuration needed.
4. Hit **Deploy**. Done. ☪️

It also works on **Netlify**, **GitHub Pages**, **Cloudflare Pages**, or literally any static host.
